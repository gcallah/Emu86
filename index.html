Emu86 Assembly Language Emulator

Emu86 is a browser-based interpreter for x86 assembly language written in Python, using the Django framework. It allows code-stepping, displays registers, flags, memory, and the stack, and provides online help for all instructions. It is intended as a learning tool for introducing students to assembler.
